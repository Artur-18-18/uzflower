# 📊 Platform Comparison: Railway vs Render

Сравнение платформ для деплоя UzFlower

---

## 🏆 Быстрое сравнение

| Критерий | Railway | Render |
|----------|---------|--------|
| **Free Tier** | $5 кредитов | 750 часов |
| **Min Price** | $5/месяц | $7/месяц |
| **RAM (Free)** | 1GB | 512MB |
| **CPU (Free)** | Shared | Shared |
| **PostgreSQL** | ✅ Включена | ✅ Включена |
| **Auto-deploy** | ✅ Да | ✅ Да |
| **SSL** | ✅ Автоматически | ✅ Автоматически |
| **Custom Domain** | ✅ Да | ✅ Да |
| **Docker** | ✅ Поддержка | ✅ Поддержка |
| **Sleep Mode** | ❌ Нет | ⚠️ Да (Free) |

---

## 🎯 Рекомендации

### Выберите **Railway** если:

- ✅ Нужно больше RAM (1GB vs 512MB)
- ✅ Не хотите sleep mode
- ✅ Любите простой интерфейс
- ✅ Нужна поддержка Docker
- ✅ Хотите $5 кредитов бесплатно

**Цена:** $5/месяц за базовый план

### Выберите **Render** если:

- ✅ Хотите полностью бесплатный тариф
- ✅ Не страшен sleep mode (15 мин)
- ✅ Нужна DDoS защита
- ✅ Хотите custom domain бесплатно
- ✅ Нужна лучшая документация

**Цена:** $7/месяц чтобы не спал
