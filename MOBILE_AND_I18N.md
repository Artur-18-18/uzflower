# Мобильная адаптация и переводы UzFlower

## 📱 Мобильная адаптация

### Поддерживаемые разрешения

| Устройство | Ширина | Статус |
|------------|--------|--------|
| iPhone SE | 320px | ✅ Полная поддержка |
| iPhone 12/13 | 390px | ✅ Полная поддержка |
| iPhone 14 Pro Max | 430px | ✅ Полная поддержка |
| Android (средний) | 360-412px | ✅ Полная поддержка |
| Планшет | 768-1024px | ✅ Полная поддержка |
| Desktop | > 1024px | ✅ Полная поддержка |

### Адаптивные компоненты

#### Header
- **Desktop**: Полная навигация, поиск, кнопки
- **Mobile**: Компактный header, бургер-меню, нижняя панель навигации

#### Hero Banner
- **Desktop**: Большой заголовок (6xl), две кнопки в ряд
- **Mobile**: Уменьшенный заголовок (2.5rem), кнопки вертикально

#### Категории
- **Desktop**: 6 колонок
- **Планшет**: 3 колонки
- **Mobile**: 2 колонки

#### Товары
- **Desktop**: 3-4 колонки
- **Планшет**: 2-3 колонки
- **Mobile**: 2 колонки (на очень маленьких - 1 колонка)

#### Преимущества
- **Desktop**: 4 колонки в ряд
- **Mobile**: 1-2 колонки

#### Нижняя навигационная панель (Mobile)
- Фиксированная панель внизу экрана
- 4 основные кнопки: Главная, Поиск, Избранное, Корзина
- Анимированное меню с эффектом волны

### CSS файлы

| Файл | Описание |
|------|----------|
| `responsive.css` | Основные медиа-запросы для всех устройств |
| `mobile-menu.css` | Стили для мобильного нижнего меню |
| `style.css` | Базовые стили + переключатель языка |

### Breakpoints

```css
/* Очень маленькие экраны (< 360px) */
@media (max-width: 360px) { }

/* iPhone SE / Small phones (< 375px) */
@media (max-width: 375px) { }

/* Mobile (< 768px) */
@media (max-width: 768px) { }

/* Tablet (768px - 1024px) */
@media (min-width: 769px) and (max-width: 1024px) { }

/* Landscape mode */
@media (max-height: 500px) and (orientation: landscape) { }
```

```json
  "validation": { }        // Валидация

```

### Использование

#### В HTML
```html
<!-- Простой перевод -->
<h1 data-i18n="hero.title">Весенняя Коллекция</h1>

<!-- Placeholder -->
<input data-i18n-placeholder="header.search_placeholder">

<!-- Title attribute -->
<button data-i18n-title="cart.view">Просмотр</button>
```

#### В JavaScript
```javascript
// Получить перевод
const text = i18n.t('products.add_to_cart');

// Перевод с параметрами
const msg = i18n.t('validation.min_value', { min: 10000 });

// Переключить язык
i18n.setLang('uz');

// Перевести всю страницу
i18n.translatePage();

// Событие смены языка
window.addEventListener('languageChanged', (e) => {
    console.log('Новый язык:', e.detail.lang);
});
```

### Переключатель языка

Расположение:
- **Сайт**: В header, справа от иконок корзины/избранного
- **Админка**: В header, рядом с заголовком

Внешний вид:
```
🇷🇺 🇺🇿
```

Активный язык подсвечивается розовой рамкой.

### Хранение выбора

Язык сохраняется в `localStorage`:
```javascript
localStorage.setItem('uzflower_lang', 'uz');
```

При загрузке страницы язык автоматически восстанавливается.

## 🔧 Обновление переводов

### Добавить новый ключ

1. Откройте `locales/ru.json` и `locales/uz.json`
2. Добавьте новый ключ в соответствующую секцию
3. Скопируйте файлы в `static/js/i18n/`

```bash
copy locales\ru.json static\js\i18n\ru.json
copy locales\uz.json static\js\i18n\uz.json
```

### Использовать в шаблоне

```html
<span data-i18n="my.new.key">Текст</span>
```

## 🐛 Известные ограничения

1. **Динамический контент** (товары из API) переводится через JavaScript в `app.js`
2. **Изображения с текстом** не переводятся автоматически
3. **Email уведомления** требуют отдельной системы переводов
